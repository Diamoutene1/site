Mes Projets

Projet de Licence 1 : Création de Jeux en Python

Description : Pendant ma première année en informatique, j'ai développé deux jeux classiques en utilisant le langage Python avec l'interface Tkinter :

  • Le Jeu de la Vie : Un automate cellulaire où les cellules évoluent en fonction de règles simples, simulant des phénomènes naturels complexes.
  • Le Jeu du Serpent : Un jeu où le joueur contrôle un serpent qui grandit à chaque fois qu'il mange, et où le but est de ne pas se heurter aux bords ou à son propre corps.

Technologies utilisées : Python, Tkinter (interface graphique).

Projet de Licence 1 : JavaScript Asynchrone et Cartographie

Description : Ce projet consiste à afficher la localisation d'une image sur une carte, puis à ajouter sur la carte les images Flickr qui sont géolocalisées sur cette région.

Technologies utilisées : JavaScript, API Flickr, API de cartographie.

Projet de Baccalauréat : Système d'Arrosage Automatique

Description : Lors de mon baccalauréat en électronique, j'ai conçu un système d'arrosage automatique contrôlé par SMS. Le système permet d'arroser les plantes en envoyant un SMS à un numéro dédié. Grâce à un module GSM, le système reçoit les instructions par SMS et active les relais pour déclencher l'arrosage selon les besoins.

Technologies utilisées : Microcontrôleur, Module GSM, Relais, Capteurs d'humidité du sol.
